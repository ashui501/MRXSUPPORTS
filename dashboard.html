Command 1: Update and Upgrade

This command updates and upgrades all existing packages.

pkg update && upgrade -y

Command 2: Termux Storage Setup

This command allows Termux to access your device's internal and external storage.

termux-setup-storage

Command 3: List Files

Displays the contents of the current directory.

ls

Command 4: Copy Installer.sh to Home Directory

This command copies the installer script to the home directory. Replace <your_installer.sh_file_path> with the actual path.

cp <your_installer.sh_file_path> $HOME/

Example

Command 5: Run Installer

This command runs the Installer.sh script.

bash Installer.sh

Command 6: Change Directory to BeEF

This command navigates to the BeEF directory after installation.

cd beef

Command 7: Install Dependencies using Bundler

This command installs the required Ruby gems using Bundler. Ensure that you have bundler installed first.

bundle install

Command 8: Start BeEF Framework

This command starts the BeEF framework.

./beef

Command 9: Access BeEF Web Interface

After starting BeEF, you can access the web interface using the following URL. Make sure BeEF is running before you visit this URL:

http://localhost:3000/ui/panel
